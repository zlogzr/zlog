把脑子里的东西写下来，是我能找到的最好的学习方式。

## 为什么写

- **输出倒逼输入**：要讲清楚一件事，你得先真的搞懂它。
- **对抗遗忘**：三个月后的自己，约等于另一个人。
- **复利**：每一篇都是可被搜索、可被链接的资产。

## 怎么写

我把内容分成两类：

1. **博客**（你正在看的）—— 带时间线的文章和思考。
2. **知识库** —— 按主题沉淀、会反复修订的知识点。

下面试一段代码高亮：

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

> 开始，比完美更重要。
